사용 방법

발급 후 강사님과 학생에게 각각 다른 값을 전달합니다.

안내
강사님: `/admin`에서 라이선스 키로 로그인합니다.
학생: `/student`에서 아이디, 비밀번호와 함께 선생님 코드를 입력합니다.
나: 기존 `ADMIN_PASSWORD`로 로그인하면 전체 기본 관리자 공간을 사용합니다.

주의

라이선스는 서명된 키라서 발급 후에는 키 자체가 권한이 됩니다.

이 파일에서 기록을 삭제해도 이미 발급한 라이선스가 자동 폐기되지는 않습니다.
최근 접속 기록은 현재 이 관리 파일에서 수동으로 남기는 기록입니다. 실제 접속 시간을 자동으로 모으려면 서버 저장형 라이선스 관리로 확장해야 합니다.
나중에 “회수/정지”까지 필요하면 서버에 라이선스 목록을 저장하는 방식으로 확장하면 됩니다.
